搜尋引擎優化

An essential part of good web design and development is SEO. Well-structured code is the key to ensuring that a web page not only gets properly indexed by search engines, but made accessible to those with limited web capabilities as well.

認識 SEO 最佳實務

  • 列印用 CSS 的最佳實務
  • 網站與程式應遵守瀏覽器解析度準則。
  • 網站與程式應遵守瀏覽器支援與測試中提到的相容需求。
  • 多了解網頁無障礙的最佳實務，例如 Section 508 及 WCAG 標準：
    • http://www.section508.gov
    • http://www.w3.org/TR/WCAG20/.

可索引

我們必須使用在 JavaScript 及 CSS 關閉的情形下，可讀、符合邏輯的語意標籤撰寫網頁。所有網頁的內容都必須放進 HTML 中，不以 iframe 或 JavaScript 來讀入可供索引的內容。

這可以讓網頁順利被搜尋引擎列入索引，也方便使用者將連結開啟於新分頁或新視窗中。

優化

title 標籤中應包括該頁面的關鍵字，每頁的標題都不同。內文標題（h1、h2 等等）應為文件的綱要、並含有該頁最重要的關鍵字。網址應可易讀達意，包含該頁的關鍵字：

http://domain.com/mens-shoes/basketball/jordan/jordan-mens-ajf-6-basketball-shoe/

Google 的 SEO 報告

Google 的 SEO 報告是提供 Google 產品開發團隊參考、以便用簡單容易採納的優化步驟改進產品網頁的文件。些優化方式不只能協助搜尋引擎更瞭解網頁內容，還可以提升訪客的使用體驗。

所謂的簡單步驟包括修復 404 網頁及錯誤連結、簡化網址規劃、提供易於瞭解的標題與說明等，對訪客及搜尋引擎都有好處。
